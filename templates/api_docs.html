📡 API REST Documentation

Intégrez le téléchargement vidéo dans vos applications. Tous les endpoints retournent du JSON.

https://tubestream.app/api/v1
POST /api/v1/info Récupérer les métadonnées d'une vidéo

Récupère toutes les informations d'une vidéo ou playlist : titre, description, durée, thumbnail, et tous les formats disponibles avec leur taille exacte.

{

}

POST /api/v1/formats Formats disponibles (léger)

Version légère de /info — retourne uniquement la liste des formats avec tailles, sans les métadonnées complètes de la vidéo.

Body

{ "url": "https://youtube.com/watch?v=..." }
POST /api/v1/download Lancer un téléchargement

Démarre le téléchargement d'un format spécifique. Retourne un session_id pour suivre la progression.

Body

{
  "url": "https://youtube.com/watch?v=...",
  "format_id": "22"  // optionnel, utilise "best" si absent
}

Réponse (200)

{
  "session_id": "a1b2c3d4",
  "status": "started",
  "progress_url": "/api/v1/progress/a1b2c3d4"
}
GET /api/v1/progress/{session_id} Vérifier la progression

Interrogez cet endpoint pour suivre l'avancement du téléchargement en temps réel.

Réponse (en cours)

{
  "status": "downloading",
  "progress": 45.2,
  "message": "Downloading... 45.2%",
  "speed": "2.5 MiB/s",
  "eta": "1:30",
  "total_mb": 25.3,
  "downloaded_mb": 11.4
}

Réponse (terminé)

{
  "status": "completed",
  "progress": 100,
  "filename": "video.mp4",
  "size_mb": 24.8,
  "download_url": "/api/v1/file/a1b2c3d4/video.mp4"
}
GET /api/v1/file/{session_id}/{filename} Télécharger le fichier

Télécharge le fichier résultant du téléchargement. Le fichier est servi avec l'en-tête Content-Disposition: attachment.

GET /api/v1/sites Liste des sites supportés

Retourne la liste complète des 1000+ sites supportés par l'API.

Réponse

{
  "total": 1024,
  "sites": ["YouTube", "TikTok", "Instagram", ...]
}
POST /api/v1/cleanup/{session_id} Nettoyer une session

Supprime les fichiers temporaires d'une session de téléchargement. Appelez cet endpoint après avoir téléchargé le fichier.

💻 Exemples de code

cURL — Récupérer les formats

curl -X POST https://tubestream.app/api/v1/info \
  -H "Content-Type: application/json" \
  -d '{"url": "https://www.youtube.com/watch?v=dQw4w9WgXcQ"}'

Python — Télécharger une vidéo

import requests

# 1. Récupérer les formats
res = requests.post("https://tubestream.app/api/v1/info", json={
    "url": "https://youtube.com/watch?v=dQw4w9WgXcQ"
})
info = res.json()

# 2. Lancer le téléchargement
dl = requests.post("https://tubestream.app/api/v1/download", json={
    "url": "https://youtube.com/watch?v=dQw4w9WgXcQ",
    "format_id": info["formats"]["combined"][0]["format_id"]
})
session_id = dl.json()["session_id"]

# 3. Suivre la progression
import time
while True:
    progress = requests.get(f"https://tubestream.app/api/v1/progress/{session_id}").json()
    print(f"{progress['progress']:.1f}% — {progress['message']}")
    if progress["status"] in ("completed", "error"):
        break
    time.sleep(1)

# 4. Télécharger le fichier
if progress["status"] == "completed":
    file_url = "https://tubestream.app" + progress["download_url"]
    r = requests.get(file_url)
    with open(progress["filename"], "wb") as f:
        f.write(r.content)
    print("✅ Fichier téléchargé !")

JavaScript / Fetch — Télécharger

// Récupérer les infos
const info = await fetch("/api/v1/info", {
    method: "POST",
    headers: { "Content-Type": "application/json" },
    body: JSON.stringify({ url: "https://youtube.com/watch?v=..." })
}).then(r => r.json());

// Télécharger
const dl = await fetch("/api/v1/download", {
    method: "POST",
    headers: { "Content-Type": "application/json" },
    body: JSON.stringify({
        url: "https://youtube.com/watch?v=...",
        format_id: info.formats.combined[0].format_id
    })
}).then(r => r.json());

// Polling
const interval = setInterval(async () => {
    const p = await fetch(`/api/v1/progress/${dl.session_id}`).then(r => r.json());
    console.log(p.progress + "%");
    if (p.status === "completed") {
        clearInterval(interval);
        window.location.href = p.download_url;
    }
}, 500);
